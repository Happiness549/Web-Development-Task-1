# Personal Portfolio Website

## 📌 Project Overview
This is my personal portfolio website built using semantic HTML and CSS.  
The website showcases information about me, my skills, and ways to contact me.

The project includes:
- Home section
- About Me section
- Contact section
- Social media links
- Responsive design

---

## 🚀 Features
- Clean and modern layout
- Responsive design for mobile and desktop
- Semantic HTML structure
- Styled contact form
- Footer with social media icons
